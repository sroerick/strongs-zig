# G3592 ὅδε

## hóde

_(hod'-eh | HOH-theh | OH-thay)_

### Definition

from [[G3588]] and [[G1161]]; the same, i.e., this or that one (plural these or those); often used as a personal pronoun; he, she, such, these, thus.

- this one here, these things, as follows, thus
