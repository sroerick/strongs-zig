# G101 ἀδυνατέω

## adynatéō

_(ad-oo-nat-eh'-o | ah-thoo-na-TEH-oh | ah-thyoo-na-TAY-oh)_

### Definition

from [[G102]]; to be unable, i.e., (passively) impossible; be impossible.

- impossible
  - not to have strength, power, or ability, to be weak
  - can not be done, to be impossible
