# G4340 πρόσκαιρος

## próskairos

_(pros'-kahee-ros | PROSE-keh-rose | PROSE-kay-rose)_

### Definition

from [[G4314]] and [[G2540]]; for the occasion only, i.e., temporary; dur-(eth) for awhile, endure for a time, for a season, temporal.

- for a season
- enduring only for a while
- temporary
