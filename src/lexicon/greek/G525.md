# G525 ἀπαλλάσσω

## apallássō

_(ap-al-las'-so | ah-pahl-LAHS-soh | ah-pahl-LAHS-soh)_

### Definition

from [[G575]] and [[G236]]; to change away, i.e., release, (reflexively) remove; deliver, depart.

- to remove, release, to be removed, to depart
- to set free, deliver
