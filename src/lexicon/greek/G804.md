# G804 ἀσφαλής

## asphalḗs

_(as-fal-ace' | ah-sfa-LASE | ah-sfa-LASE)_

### Definition

from [[G1]] (as a negative particle) and sphallo (to "fail"); secure (literally or figuratively); certain(-ty), safe, sure.

- firm (that which can be relied on)
- certain, true
- suited to confirm
