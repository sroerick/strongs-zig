# G5616 ὡσεί

## hōseí

_(ho-si' | hoh-SEE | oh-SEE)_

### Definition

from [[G5613]] and [[G1487]]; as if; about, as (it had been, it were), like (as).

- as it were, (had been), as though, as, like as, like
- about, nearly
  - before numerals
  - before a measure of time
