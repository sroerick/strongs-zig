# G2553 κακοπαθέω

## kakopathéō

_(kak-op-ath-eh'-o | ka-koh-pa-THEH-oh | ka-koh-pa-THAY-oh)_

### Definition

from the same as [[G2552]]; to undergo hardship; be afflicted, endure afflictions (hardness), suffer trouble.

- to suffer (endure) evils (hardships, troubles)
- to be afflicted
