# H4385 מִכְתָּב / מכתב

## miktâb

_(mik-tawb' | mik-TAWB | meek-TAHV)_

### Definition

masculine noun; from [[H3789]]; a thing written, the characters, or a document (letter, copy, edict, poem); writing.

- writing, thing written
    - handwriting
    - thing written
    - writing
