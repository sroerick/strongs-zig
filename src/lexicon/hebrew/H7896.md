# H7896 שִׁית / שית

## shîyth

_(sheeth | sheet | sheet)_

### Definition

verb; a primitive root; to place (in a very wide application); apply, appoint, array, bring, consider, lay (up), let alone, [[lexicon-key|×]] look, make, mark, put (on), [[lexicon-key|+]] regard, set, shew, be stayed, [[lexicon-key|×]] take.

- to put, set
    - (Qal)
        - to put, lay (hand upon)
        - to set, station, appoint, fix, set mind to
        - to constitute, make (one something), make like, perform
        - to take one's stand
        - to lay waste
    - (Hophal) to be imposed, be set upon
