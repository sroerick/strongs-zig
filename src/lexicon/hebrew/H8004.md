# H8004 שָׁלֵם / שלם

## Shâlêm

_(shaw-lame' | shaw-LAME | sha-LAME)_

### Definition

proper locative noun; peace; the same as [[H8003]]; peaceful; Shalem, an early name of Jerusalem; Salem.

- the place of which Melchizedek was king
    - most Jewish commentators affirm that it is the same as Jerusalem
