# H6009 עָמַק / עמק

## ʻâmaq

_(aw-mak' | aw-MAHK | ah-MAHK)_

### Definition

verb; a primitive root; to be (causatively, make) deep (literally or figuratively); (be, have, make, seek) deep(-ly), depth, be profound.

- to be deep, be profound, make deep
    - (Qal) to be deep
    - (Hiphil) to make deep, make profound
