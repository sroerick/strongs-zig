# H1595 גֶּנֶז / גנז

## genez

_(gheh'-nez | ɡeh-NEDZ | ɡeh-NEZ)_

### Definition

masculine noun; from an unused root meaning to store; treasure; by implication, a coffer; chest, treasury.

- treasury, chests?
    - chests (of variegated cloth) (meaning uncertain)
    - treasury
