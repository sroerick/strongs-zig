# H3685 כְּסִיל / כסיל

## Kᵉçîyl

_(kes-eel' | keh-SEEL | keh-SEEL)_

### Definition

masculine noun; the same as [[H3684]]; any notable constellation; specifically Orion (as if a burly one); constellation, Orion.

- constellation, Orion
    - Orion, the constellation
    - constellation (general)
