# H7164 קָרַס / קרס

## qâraç

_(kaw-ras' | kaw-RAHS | ka-RAHS)_

### Definition

verb; a primitive root; properly, to protrude; used only as denominative from [[H7165]] (for alliteration with [[H7167]]); to hunch, i.e., be hump-backed; stoop.

- (Qal) to bend down, stoop, crouch
