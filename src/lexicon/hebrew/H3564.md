# H3564 כּוּר / כור

## kûwr

_(koor | koor | koor)_

### Definition

masculine noun; from an unused root meaning properly, to dig through; a pot or furnace (as if excavated); furnace.

- furnace, forge, smelting furnace or pot (noun masculine)
- (Qal) to bore, pierce, dig, hew (verb)
