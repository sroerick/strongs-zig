# H481 אָלַם / אלם

## ʼâlam

_(aw-lam' | aw-LAHM | ah-LAHM)_

### Definition

verb; a primitive root; to tie fast; hence (of the mouth) to be tongue-tied; bind, be dumb, put to silence.

- to bind
    - (Niphal)
        - to be dumb
        - to be bound
    - (Piel) binding (participle)
