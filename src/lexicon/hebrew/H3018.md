# H3018 יְגִיעַ / יגיע

## yᵉgîyaʻ

_(yeg-ee'-ah | yeh-ɡEE-ah | yeh-ɡEE-ah)_

### Definition

masculine noun; from [[H3021]]; toil; hence, a work, produce, property (as the result of labor); labour, work.

- toil, work
- product, produce, acquired property (as a result of work)
