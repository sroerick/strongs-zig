# H5459 סְגֻלָּה / סגלה

## çᵉgullâh

_(seg-ool-law' | seh-ɡool-LAW | seh-ɡoo-LA)_

### Definition

feminine noun; feminine passive participle of an unused root meaning to shut up; wealth (as closely shut up); jewel, peculiar (treasure), proper good, special.

- possession, property
    - valued property, peculiar treasure
    - treasure
