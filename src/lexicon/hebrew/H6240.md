# H6240 עָשָׂר / עשר

## ʻâsâr

_(aw-sawr' | aw-SAWR | ah-SAHR)_

### Definition

noun; for [[H6235]]; ten (only in combination), i.e., -teen; also (ordinal) -teenth; (eigh-, fif-, four-, nine-, seven-, six-, thir-)teen(-th), [[lexicon-key|+]] eleven(-th), [[lexicon-key|+]] sixscore thousand, [[lexicon-key|+]] twelve(-th).

- ten, -teen (in combination with other numbers)
    - used only in combination to make the numbers 11-19
