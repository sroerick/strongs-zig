# H7412 רְמָה / רמה

## rᵉmâh

_(rem-aw' | reh-MAW | reh-MA)_

### Definition

verb; corresponding to [[H7411]]; to throw, set, (figuratively) assess; cast (down), impose.

- to cast, throw
    - (Peal)
        - to cast
        - to throw
            - of tribute (figuratively)
    - (Peil)
        - to be cast
        - to be placed, be set
    - (Ithpeel) to be cast
