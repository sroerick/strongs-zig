# H6001 עָמֵל / עמל

## ʻâmêl

_(aw-male' | aw-MALE | ah-MALE)_

### Definition

adjective; from [[H5998]]; toiling; concretely, a laborer; figuratively, sorrowful; that laboureth, that is a misery, had taken (labour), wicked, workman.

- labourer, sufferer, wretched one (noun masculine)
    - labourer, workman
    - sufferer
- toiling (adjectival verb)
