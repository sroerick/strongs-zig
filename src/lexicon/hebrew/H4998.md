# H4998 נָאָה / נאה

## nâʼâh

_(naw-aw' | naw-AW | na-AH)_

### Definition

verb; a primitive root; properly, to be at home, i.e., (by implication) to be pleasant (or suitable), i.e., beautiful; be beautiful, become, be comely.

- to be comely, be beautiful, be befitting
    - (Pilpel)
        - to be comely
        - to be befitting
