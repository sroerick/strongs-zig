# H5810 עָזַז / עזז

## ʻâzaz

_(aw-zaz' | aw-DZAHDZ | ah-ZAHZ)_

### Definition

verb; a primitive root; to be stout (literally or figuratively); harden, impudent, prevail, strengthen (self), be strong.

- to be strong
    - (Qal) to be strong, prevail
    - (Hiphil) to make firm, strengthen
