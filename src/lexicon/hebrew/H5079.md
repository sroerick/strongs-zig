# H5079 נִדָּה / נדה

## niddâh

_(nid-daw' | nid-DAW | nee-DA)_

### Definition

feminine noun; from [[H5074]]; properly, rejection; by implication, impurity, especially personal (menstruation) or moral (idolatry, incest); [[lexicon-key|×]] far, filthiness, [[lexicon-key|×]] flowers, menstruous (woman), put apart, [[lexicon-key|×]] removed (woman), separation, set apart, unclean(-ness, thing, with filthiness).

- impurity, filthiness, menstruous, set apart
    - impurity
        - of ceremonial impurity
        - of menstruation
    - impure thing (figuratively)
        - of idolatry, immorality
