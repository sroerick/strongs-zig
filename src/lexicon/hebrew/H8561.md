# H8561 תִּמֹּר / תמר

## timmôr

_(tim-more' | tim-MORE | tee-MORE)_

### Definition

feminine noun; (plural or (feminine) תִּמֹּרָה; (singular and plural), from the same root as [[H8558]]; (architectural) a palm-like pilaster (i.e., umbellate); palm tree.

- palm tree figure (as ornament)
