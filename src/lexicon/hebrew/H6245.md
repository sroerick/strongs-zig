# H6245 עָשַׁת / עשת

## ʻâshath

_(aw-shath' | aw-SHAHT | ah-SHAHT)_

### Definition

verb; a primitive root; probably to be sleek, i.e., glossy; hence (through the idea of polishing) to excogitate (as if forming in the mind); shine, think.

- (Qal) to be smooth, be shiny, gleam
- (Hithpael) to think
