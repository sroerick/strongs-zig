# H5708 עֵד / עד

## ʻêd

_(ayd | ade | ade)_

### Definition

feminine noun; from an unused root meaning to set a period (compare [[H5710]], [[H5749]]); the menstrual flux (as periodical); by implication (in plural) soiling; filthy.

- menstruation
    - filthy rag, stained garment (figuratively of best deeds of guilty people)
