# H328 אַט / אט

## ʼaṭ

_(at | at | at)_

### Definition

adverb; from an unused root perhaps meaning to move softly; (as an adverb) gently; charmer.

- gentleness, softness
    - gently
    - flowing
    - softly
    - enchanter, necromancer (substantive)
