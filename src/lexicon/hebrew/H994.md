# H994 בִּי / בי

## bîy

_(bee | bee | bee)_

### Definition

particle of speech; perhaps from [[H1158]] (in the sense of asking); properly, a request; used only adverbially (always with 'my Lord'); oh that!; with leave, or if it please; alas, O, oh.

- if it please, pray excuse me, excuse me please
    - used to introduce an entreaty or request
