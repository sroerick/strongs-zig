# H599 אָנַף / אנף

## ʼânaph

_(aw-naf' | aw-NAHP | ah-NAHF)_

### Definition

verb; a primitive root; to breathe hard, i.e., be enraged; be angry (displeased).

- to be angry, to be displeased, to breathe hard
    - (Qal) to be angry (of God)
    - (Hithpael) to be angry (always of God)
