# H2300 חָדַד / חדד

## châdad

_(khaw-dad' | haw-DAHD | ha-DAHD)_

### Definition

verb; a primitive root; to be (causatively, make) sharp or (figuratively) severe; be fierce, sharpen.

- to be sharp, be alert, be keen
    - (Qal) to be sharp, be keen
    - (Hiphil) to sharpen
    - (Hophal) to be sharpened
