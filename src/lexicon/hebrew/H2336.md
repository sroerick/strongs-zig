# H2336 חוֹחַ / חוח

## chôwach

_(kho'-akh | HOH-ah | HOH-ak)_

### Definition

masculine noun; from an unused root apparently meaning to pierce; a thorn; by analogy, a ring for the nose; bramble, thistle, thorn.

- thorn, brier, bramble, thornbush, thicket
- hook, ring, fetter
