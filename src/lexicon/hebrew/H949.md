# H949 בּוֹצֵץ / בוצץ

## Bôwtsêts

_(bo-tsates' | boh-TSAYTS | boh-TSAYTS)_

### Definition

proper locative noun; surpassing white, glistening; from the same as [[H948]]; shining; Botsets, a rock near Michmash; Bozez.

- a rock formation near Michmash by which Jonathan approached the Philistine troop
