# H5553 סֶלַע / סלע

## çelaʻ

_(seh'-lah | seh-LA | seh-LA)_

### Definition

masculine noun; from an unused root meaning to be lofty; a craggy rock, literally or figuratively (a fortress); (ragged) rock, stone(-ny), strong hold.

- crag, cliff, rock
    - crag, cliff
    - as stronghold of Jehovah, of security (figuratively)
