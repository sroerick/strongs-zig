# H6631 צֶאֱצָא / צאצא

## tseʼĕtsâʼ

_(tseh-ets-aw' | tseh-eh-TSAW | tseh-ay-TSA)_

### Definition

masculine noun; from [[H3318]]; issue, i.e., produce, children; that which cometh forth (out), offspring.

- offspring, produce, issue
    - offspring (of men)
    - produce (of earth)
    - descendants (metaphorical)
