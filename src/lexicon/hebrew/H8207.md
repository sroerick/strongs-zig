# H8207 שְׁפִיפֹן / שפיפן

## shᵉphîyphôn

_(shef-ee-fone' | sheh-pee-PONE | sheh-fee-FONE)_

### Definition

masculine noun; from an unused root meaning the same as [[H7779]]; a kind of serpent (as snapping), probably the cerastes or horned adder; adder.

- horned snake
- ([[lexicon-key|CLBL]]) a serpent, perhaps an adder or horned snake
