# H8564 תַּמְרוּר / תמרור

## tamrûwr

_(tam-roor' | tahm-ROOR | tahm-ROOR)_

### Definition

masculine noun; from the same root as [[H8558]]; an erection, i.e., pillar (probably for a guide-board); high heap.

- guideposts, markers, signposts, high heap
    - meaning doubtful
