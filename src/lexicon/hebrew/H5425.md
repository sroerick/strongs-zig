# H5425 נָתַר / נתר

## nâthar

_(naw-thar' | naw-TAHR | na-TAHR)_

### Definition

verb; a primitive root; to jump, i.e., be violently agitated; causatively, to terrify, shake off, untie; drive asunder, leap, (let) loose, [[lexicon-key|×]] make, move, undo.

- to start up, tremble, shake, spring up
    - (Qal) to start up
    - (Piel) to leap
    - (Hiphil) to cause to start up
- to loose, let loose, undo, be free, be loose
    - (Hiphil)
        - to unfasten, loosen
        - to set free, unbind
